**VentureSmith is an AI-powered web application designed to be the ultimate co-founder for aspiring entrepreneurs. It transforms a simple business idea into a comprehensive, venture-ready blueprint, automating everything from business planning to market research and prototype generation.**

This project was built for a hackathon, showcasing the power of modern AI and serverless technologies to accelerate innovation.

## 🌟 Key Features

VentureSmith is packed with AI-driven tools to cover every aspect of early-stage startup development:

### 📈 Strategy & Planning
- **📝 Business Plan Generation:** Creates a full business plan from a single idea prompt.
- **🎯 Mission & Vision Crafting:** Defines a strong foundation for your brand.
- **🗺️ Development Roadmap:** Lays out a technical timeline and milestones.
- **✅ Due Diligence Checklist:** Prepares you for investor scrutiny.
- **💡 Idea Brainstorming & A/B Testing:** Generates creative ideas and testing strategies.

### 📊 Market & Customer Analysis
- **🌐 AI Market Research:** Uses Firecrawl to scrape and analyze web data for market trends and competitor insights.
- **👥 Customer Persona Creation:** Defines your ideal customer profiles.
- **🔍 Competitor Matrix:** Generates a visual breakdown of the competitive landscape.
- **📈 Growth Metrics Identification:** Helps you define KPIs to track success.

### 🛠️ Product & Engineering
- **💻 AI Wireframe Generator:** Creates visual mockups for your web application.
- **⚙️ Tech Stack Suggester:** Recommends the best technologies for your project.
- **🗃️ Database Schema Generation:** Designs the data model for your application.
- **🔌 API Endpoint Definition:** Drafts the API structure for your backend.
- **☁️ Cloud Cost Estimation:** Provides a preliminary budget for cloud infrastructure.

### 🚀 Marketing & Launch
- **📢 Press Release Drafting:** Writes professional announcements for your launch.
- **✍️ Marketing Copy Generation:** Creates compelling copy for ads, landing pages, and social media.
- **SEO Strategy:** Generates a foundational SEO plan to improve online visibility.
- **🚀 Product Hunt Launch Kit:** Prepares materials for a successful Product Hunt debut.

### 💬 Interactive Tools
- **🤖 VentureChatbot:** An AI assistant trained on your business context to answer questions and provide guidance.
- **🧑‍🏫 AI Pitch Coach:** Provides feedback and suggestions to refine your investor pitch.
- **🏢 SmithWorkspace:** A centralized dashboard to manage and view all your generated assets.

## ⚙️ How It Works

1.  **Sign Up & Submit Idea:** A user signs up and enters their core business idea.
2.  **AI Processing:** VentureSmith's Convex backend orchestrates calls to the Gemini API for analysis and content generation.
3.  **Web Scraping:** For market research, the backend triggers a Firecrawl job to gather real-time data from the web.
4.  **Generate Assets:** The user can then navigate through the SmithWorkspace to generate various documents and analyses—from business plans to wireframes.
5.  **Review & Refine:** All generated content is displayed in a clean, user-friendly interface for review and use.

## 🚀 Getting Started

To run VentureSmith on your local machine, follow these steps:

### Prerequisites

- Node.js (v18 or later)
- An account with [Convex](https://convex.dev)
- API keys for:
  - Google Gemini
  - Firecrawl
  - Resend

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/zaikaman/venturesmith.git
    cd venturesmith
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up Convex:**
    - Login to Convex:
      ```bash
      npx convex dev
      ```
    - Follow the CLI prompts to link this project to your Convex account.

4.  **Configure Environment Variables:**
    - Create a `.env.local` file in the root directory by copying the example file:
      ```bash
      cp .env.example .env.local
      ```
    - Open `.env.local` and add your credentials. The `VITE_CONVEX_URL` will be provided in the output of the `npx convex dev` command.

    ```env
    # Convex deployment URL from `npx convex dev`
    VITE_CONVEX_URL="https://your-project.convex.cloud"

    # API Keys
    GEMINI_API_KEY="your_gemini_api_key"
    FIRECRAWL_API_KEY="your_firecrawl_api_key"
    RESEND_API_KEY="your_resend_api_key"

    # Convex Admin Key (for backend functions)
    CONVEX_AUTH_PRIVATE_KEY="-----BEGIN EC PRIVATE KEY-----
...
-----END EC PRIVATE KEY-----"
    JWT_ISSUER_URL="https://your-convex-project.us.convex.cloud"
    ```

5.  **Run the development server:**
    - The `npx convex dev` command from step 3 also runs the backend. In a new terminal, start the frontend:
    ```bash
    npm run dev
    ```
    - Open your browser and navigate to `http://localhost:5173`.

## 📂 Project Structure

The codebase is organized into logical directories:

```
/
├── convex/             # All Convex backend functions (queries, mutations, actions)
│   ├── gemini.ts       # Logic for interacting with the Gemini API
│   ├── firecrawl.ts    # Logic for web scraping actions
│   ├── schema.ts       # Database schema definition
│   └── ...             # Other backend files for startups, users, etc.
├── src/
│   ├── components/     # Reusable React components
│   │   ├── pages/      # Components for each AI tool/feature
│   │   └── chatbot/    # The VentureChatbot implementation
│   ├── services/       # Frontend services (e.g., Gemini service wrapper)
│   ├── contexts/       # React contexts (Theme, etc.)
│   ├── App.tsx         # Main application component and router
│   └── index.tsx       # Application entry point
├── public/             # Static assets
└── package.json        # Project dependencies and scripts
```
